# FORGE — Formal Output Regulation for Generative Engineering

A unified repository serving as both a portfolio tool and Master's thesis infrastructure. Contains a parametric battery cell design engine, neuro-symbolic LLM supervision architecture (AXIOM), REST API for enterprise integration, ML surrogate modelling skeleton, and thesis evaluation infrastructure.

Project is under active development.
